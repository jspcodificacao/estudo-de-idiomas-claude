# Frontend - Estudo de Idiomas

Aplicação web desenvolvida com Vite, React, TypeScript e Tailwind CSS.

## Funcionalidades

- Editar Prompts
- Mudar Base de Conhecimento
- Navegar no Histórico
- Editar Frases do Diálogo

## Tecnologias

- **Vite** - Build tool e dev server
- **React 18** - Biblioteca UI
- **TypeScript** - Tipagem estática
- **Tailwind CSS** - Framework CSS utilitário
- **React Router** - Roteamento

## Instalação

```bash
npm install
```

## Executar em modo de desenvolvimento

```bash
npm run dev
```

A aplicação será iniciada na porta configurada em `FRONTEND_PORT` no arquivo `.env` (padrão: 5173).

## Build para produção

```bash
npm run build
```

## Preview da build de produção

```bash
npm run preview
```

## Configuração

A porta do frontend é configurada através da variável de ambiente `FRONTEND_PORT` no arquivo `.env` na raiz do projeto.
